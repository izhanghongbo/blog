使用virtualenv

安装virtualenv

1

pip install virtualenv

创建运行环境

1

virtualenv venv

激活环境

1
2

cd venv
source ./bin/activate

退出环境

1

deactivate

使用venv

激活

1
2

python3 -m venv  .
source bin/activate

退出

1

source bin/activate

为Jupiter设置虚拟环境

在虚环境中安装ipykernel

1

pip install ipykernel

为Jupiter设置环境

1

python -m ipykernel install --user --name=learn

运行Jupiter

1

jupyter notebook

在Jupiter的Kernel->Change Kernel中切换内核为learn

jupyter notebook --generate-config
